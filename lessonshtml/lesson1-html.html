Lekce 1: Co je HTML, jak vzniklo a k čemu slouží?

1.1 Co je vlastně webová stránka?

Než pochopíme HTML, musíme pochopit, co je webová stránka. Webová stránka je textový soubor uložený na serveru, který si stáhne prohlížeč (Chrome, Firefox, Safari) a vykreslí ho jako vizuální stránku. Prohlížeč ale nerozumí „hezkému designu“, rozumí pouze značkám (tagům). A právě tyto značky jsou jazyk HTML.

1.2 HTML jako značkovací jazyk

    HTML není programovací jazyk.

    Neobsahuje:

  • smyčky
  • podmínky
  • výpočty
  • algoritmy

    • HTML pouze říká:

  • „Toto je nadpis.“
  • „Toto je odstavec.“
  • „Toto je obrázek.“
  • „Toto je odkaz.“

    • Jinými slovy: HTML popisuje strukturu dokumentu.

1.3 Co znamená „HyperText“?

    Hypertext znamená text, který obsahuje odkazy na jiné dokumenty

    Například:

  • klikneš na odkaz
  • otevře se jiná stránka
  • můžeš přecházet mezi informacemi

    • Tento princip je základem internetu. Bez hypertextu by internet byl jen soubor izolovaných dokumentů.

1.4 Historie HTML podrobněji

  • Rok 1989 – Tim Berners-Lee navrhuje systém propojených dokumentů.
  • 1991 – vzniká první verze HTML.

    • První HTML obsahovalo jen:

  • nadpisy
  • odstavce
  • odkazy

    • ostupně přibylo:

  • obrázky (1993)
  • tabulky
  • formuláře
  • CSS integrace
  • JavaScript podpora

    • HTML5 (2014) přineslo:

  • "video"
  • "audio"
  • "canvas"
  • nové formulářové prvky
  • sémantické značky
  • Dnes je HTML standard spravovaný organizací W3C.

1.5 Struktura HTML dokumentu do hloubky

    Každý HTML dokument má povinnou základní strukturu:

  • Říká prohlížeči, že používáme moderní HTML5.
  • Kořenový element (root element).

    • Obsahuje:

  • metadata
  • název stránky
  • odkazy na CSS
  • skripty
  • SEO informace

    • Obsahuje vše, co uživatel vidí.

1.6 Jak prohlížeč čte HTML?

  1. Stáhne soubor.
  2. Čte ho odshora dolů.
  3. Vytvoří DOM strom.
  4. Zobrazí stránku.

    5. Pokud je chyba:

  • prohlížeč se ji pokusí opravit.
  • HTML je tolerantní k chybám.
  • To je důvod, proč se někdy stránka zobrazí i při špatném kódu.

  • "h1"
