Lekce 2: Text, nadpisy a vnitřní struktura obsahu

2.1 Hierarchie obsahu

    HTML funguje jako strom:

  • rodič
  • dítě
  • sourozenec

    • Například:

<section>
<h2>Novinky</h2>
<p>Text článku</p>
</section>

    Zde:

  • "section" je rodič
  • "h2" a "p" jsou děti

2.2 Význam nadpisů

    Nadpisy nejsou jen větší text.

    Používají se k:

  • vytvoření struktury
  • navigaci pomocí čteček
  • SEO optimalizaci

    • Správně:

<h1>Hlavní téma</h1>
<h2>Podtéma</h2>
<h3>Detail</h3>

Špatně:

<h1>Velký text</h1>
<h1>Další velký text</h1>

2.3 Inline vs Block prvky

    Block prvky:

  • zabírají celý řádek
  • např. "div", "p", "h1""/p"

    • Inline prvky:

  • jsou uvnitř textu
  • např. "span", "strong", "a"

    • To je důležité pro pochopení CSS později.

Inline vs Block prvky

2.4 Sémantické HTML

Dříve se používaly jen prvky.

Dnes máme:

<header>
<nav>
<main>
<article>
<aside>
<footer>

Tyto prvky dávají význam.

Například:

<article>
<h2>Název článku</h2>
<p>Obsah článku</p>
</article>

Prohlížeč i vyhledávač ví, že jde o článek.

2.5 Seznamy do detailu

Neuspořádaný seznam (ul) používá tečky.

Uspořádaný seznam (ol) používá čísla.

Existuje i:

<dl>
<dt>Pojem</dt>
<dd>Definice</dd>
</dl>

To je definiční seznam.
